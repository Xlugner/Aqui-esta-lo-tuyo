# Product Overview

## What is this?

This is a modern e-commerce frontend for an online store. It's a customer-facing web application that allows users to browse products, view categories, manage a shopping cart, and proceed to checkout.

## Key Features

- **Product Catalog**: Browse products organized by categories
- **Shopping Cart**: Add/remove items, adjust quantities, view totals in multiple currencies (CUP/USD)
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Admin Panel**: Manage products, categories, hero section, and store configuration
- **Multi-Currency Support**: Display prices in both CUP and USD
- **Performance Optimized**: Static rendering with Astro for fast page loads
- **SEO Ready**: Optimized for search engines

## Target Users

- **Customers**: Browse and purchase products
- **Admins**: Manage store content, products, and configuration

## Business Context

- Uses Supabase for database, authentication, and storage
- Supports WhatsApp integration for customer communication
- Designed for Cuban market (CUP currency support)
